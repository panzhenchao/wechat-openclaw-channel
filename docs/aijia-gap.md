# 个人微信 IM 功能差距（AIjia vs OpenClaw）

> 对比基准：AIjia lotus-app 当前实现 vs OpenClaw 微信社区插件

## AIjia 已有的功能

- 收发文字消息（私聊 + 群聊）
- 扫码登录（iLink 方式）
- 发送图片、文件

## AIjia 还没有的功能

| 功能 | 说明 |
|------|------|
| 流式回复 | AI 回复等全部生成完才发。参考实现支持边生成边显示 |
| 多账号 | 只能登录一个微信账号 |
| 多种登录方式 | AIjia 只支持 iLink，参考实现还支持 QClaw/WorkBuddy 方式登录 |
| 白名单精细控制 | 哪些联系人/群可以触发 AI 的控制粒度较粗 |
